# 指令 {#cmd}

```json
"scripts": {
  "init": "biu init",
  "start": "biu dev --env dev",
  "build": "biu build --env prod",
  "serve": "biu serve"
},
```

:::

### emp dev

- -e, --env 部署环境 dev、test、prod 默认为 dev

### emp build

- -e, --env 部署环境 dev、test、prod 默认为 dev

### emp dts

> 根据 `config.empShare.remote` 自动同步所需类型
